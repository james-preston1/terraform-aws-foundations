# AWS Learning Plan

## Objectives
- Build foundational understanding of AWS core services
- Translate existing cloud administration and security experience into AWS
- Reinforce Infrastructure as Code concepts using Terraform

## Focus Areas
- Identity and Access Management (IAM)
- VPC networking and security boundaries
- EC2 provisioning and access patterns
- Cost awareness and monitoring basics

## Approach
- Documentation-first learning
- Incremental Terraform experimentation
- Clear notes on trade-offs and comparisons to Azure
